# D) 힌트 기능 요구사항 명세

## FR-CODE-009 수동 힌트
- 사용자가 힌트 버튼(3개)을 클릭하면 힌트 생성 요청이 수행된다.

### FR-CODE-009-1 힌트 버튼 UI
- 에디터 상단 또는 사이드에 **"🥕 힌트 (3/3)"** 버튼을 배치한다.
- 잔여 횟수를 실시간으로 표시한다.

### FR-CODE-009-2 질문 입력창
- 힌트 버튼 클릭 시 **"질문을 입력하세요..."** 텍스트 입력창이 포함된 **모달**이 열린다.

### FR-CODE-009-3 힌트 생성 중
- 힌트 요청 후 **"💡 힌트 생성 중..."** 로딩 상태를 표시한다.
- SSE로 스트리밍되는 경우 **타이핑 효과**로 점진적 표시한다.

---

## FR-CODE-010 힌트 차감
- 수동 힌트 제출 시 힌트 쿼터가 차감되며 잔여 횟수를 표시한다.
- 수동 힌트는 **'질문 입력 채팅창'**에서 질문을 입력할 수 있다.

### FR-CODE-010-1 차감 알림
- 힌트 사용 후 **"힌트가 차감되었습니다. (잔여: 2/3)"** **토스트 알림**을 표시한다.

### FR-CODE-010-2 힌트 소진
- 잔여 힌트가 **0개**일 때 힌트 버튼을 **비활성화**하고 **"힌트를 모두 사용했습니다"** **툴팁**을 표시한다.

### FR-CODE-010-3 현재 코드 전송
- 힌트 요청 시 **현재 에디터의 코드**를 함께 전송하여 **맥락 기반 힌트**를 받는다.

---

## FR-CODE-011 자동 힌트
- 백엔드 스케줄러가 **5분 단위**로 자동 분석 후 힌트를 제공한다.
- 자동 힌트가 도착하면 **메시지 모양 아이콘**의 알림이 뜬다.
- 아이콘을 클릭 시 **힌트를 받을지 말지** 여부를 버튼으로 선택할 수 있다.

### FR-CODE-011-1 알림 아이콘
- 화면 **우측 상단**에 **메시지 아이콘(💌)**을 배치한다.
- 자동 힌트 도착 시 **빨간 점(Badge)**을 표시한다.

---

## FR-CODE-012 힌트 노출
- 힌트 도착 시 **알림**과 함께 **힌트 패널**에 표시한다.
- 힌트 패널은 **터미널 창 상단의 탭 버튼**을 클릭해 이동할 수 있다.
- 힌트 노출은 **팝업 형식**으로 화면에 노출한다.

### FR-CODE-012-1 힌트 히스토리
- 힌트 패널에서 받은 **모든 힌트**를 **시간순**으로 나열한다.
- 각 힌트는 **"수동/자동" 태그**와 **시간 정보**를 포함한다.
