# 👨‍💻 코드팜(Code Farm) 시연 상세 시나리오

## 1. 메인 페이지

**화면**

- 메인 대시보드 (랭킹, 현재 학습 현황 등 노출)

**동작**

- 서비스 접속 후 전체적인 분위기 브리핑

**설명**

> “저희의 메인 페이지입니다. 코드팜은 학생들이 혼자서도 쉽고 재밌게 코딩을 배울 수 있도록 설계된 서비스입니다.”
> 

---

## 2. 커리큘럼 페이지

**화면**

- 문제 리스트 페이지 ➔ 커리큘럼 탭 클릭

**동작**

- 스크롤을 내리며 난이도별 문제 확인
- 1~5단계 커리큘럼 구조 보여주기

**설명**

- **문제 목록**
    
    > “학생들은 수준별로 구성된 문제 중 원하는 유형이나 난이도를 선택해 풀 수 있습니다.”
    > 
- **커리큘럼**
    
    > “학습 방향이 막막한 학생들을 위해 1단계 입출력부터 5단계 완전탐색까지 체계적인 커리큘럼을 제공합니다.
    > 
    > 
    > 마치 게임의 스테이지를 깨는 듯한 UI/UX로 구성되어 학생들의 거부감을 낮췄습니다.”
    > 
- **콘텐츠**
    
    > “저희는 학생들의 눈높이에 맞춘 200여 개의 문제를 직접 생성하고 검수하여 퀄리티를 높였습니다.”
    > 

---

## 3. 문제 풀이 페이지(IDE)

**화면**

- 특정 문제 상세 페이지
    - 중앙: 문제 지문
    - 우측: 코드 편집기

**동작**

- 별도 설치 없이 코드 입력 시작
- 의도적인 문법 오류(예: 반복문 오류) 발생

**설명**

- **개발 환경**
    
    > “실제 IDE가 탑재되어 별도 설치 없이 웹에서 바로 코드를 작성하고 실행할 수 있습니다.”
    > 
- **과정 중심 분석**
    
    > “지금 학생이 반복문 문법에서 어려움을 겪고 있네요.
    > 
    > 
    > 코드팜은 단순히 결과값만 보는 것이 아니라, 학생의 전체 작성 과정을 실시간으로 모니터링하여
    > 
    > ‘어떤 부분에서 막혔는지’를 분석합니다.”
    > 

---

## 4. (문제풀이 페이지)실시간 힌트 및 AI 질의응답

**화면**

- IDE 화면 좌측 힌트 채팅방

**동작**

- 도착한 힌트 확인
- 채팅창에 질문 작성

**설명**

- **힌트 제공**
    
    > “마침 힌트가 도착했네요!
    > 
    > 
    > 정답을 바로 알려주는 것이 아니라, 학생이 스스로 생각할 수 있도록 방향성을 제시해
    > 
    > 포기하지 않게 돕습니다.”
    > 
- **선생님 질문**
    
    > “만약 힌트만으로 부족하다면 ‘선생님에게 질문하기’를 통해 최대 3번까지 직접적인 도움을 받을 수 있습니다.
    > 
    > 
    > 포인트 차감 방식의 게이미피케이션 요소입니다.”
    > 

---

## 5. 결과 제출 및 학습 리포트

**화면**

- 제출 완료 메시지
- 생성된 결과 리포트 화면

**동작**

- `제출` 버튼 클릭
- 테스트 케이스 통과
- 리포트 상세 보기

**설명**

> “문제를 해결하면 내부 테스트 케이스 검사 후 리포트가 생성됩니다.
> 
> 
> 실행 시간뿐만 아니라 ‘어떤 부분에서 고민을 많이 했는지’ 등
> 
> 작성 과정에 기반한 피드백을 제공해
> 
> 실제 선생님 옆에서 지도를 받는 듯한 경험을 선사합니다.”
> 

---

## 6. 마이페이지 및 학습 이력

**화면**

- 마이페이지 (풀이 통계, 리포트 모음)

**동작**

- 과거 리포트 목록 확인, 내 정보 확인

**설명**

> “마이페이지에서는 그동안 풀었던 결과와 리포트를 모아보며,
> 
> 
> 자신이 부족한 개념이 무엇인지 스스로 확인하고 보완할 수 있습니다.”
> 

---

## 7. 게이미피케이션 (카드 뽑기 및 랭킹)

**화면**

- 카드 뽑기 페이지
- 카드 수집 목록
- 메인 랭킹판

**동작**

- 포인트 소모하여 카드 뽑기 실행
- 애니메이션이 적용된 카드 등장

**설명**

- **보상 시스템**
    
    > “공부만 하면 지루하죠.
    > 
    > 
    > 문제를 풀어 쌓은 포인트로 귀여운 카드를 뽑을 수 있습니다.
    > 
    > 보시다시피 이 카드들은 생동감 있게 움직이며 수집 욕구를 자극합니다.”
    > 
- **학습 동기 부여**
    
    > “수집한 카드는 메인 화면 랭킹에 반영됩니다.
    > 
    > 
    > 승부욕과 과시욕을 긍정적으로 활용해
    > 
    > 학생들이 중간에 포기하지 않고 학습 동기를 이어가도록 설계했습니다.”
    >
